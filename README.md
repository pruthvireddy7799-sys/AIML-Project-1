# 🎬 PICKFLIX – AI Movie Recommendation System

PICKFLIX is a premium Netflix-inspired AI movie recommendation platform built using Machine Learning and Streamlit.

The application recommends movies based on content similarity, genres, and user preferences while providing a cinematic streaming-platform-like experience.

---

# 🚀 Features

- 🎥 AI-powered movie recommendations
- 🔥 Trending movies section
- 🎭 Genre-based movie filtering
- 🎬 TMDB API integration
- ⭐ Movie ratings
- ▶ Trailer buttons
- 🖼 Movie posters and overview
- ✨ Loading skeleton animations
- 🌙 Premium Netflix-inspired cinematic UI
- ⚡ Fast and responsive design

---

# 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- NumPy
- TMDB API
- Machine Learning

---

# 🧠 Machine Learning Concepts Used

- Content-Based Filtering
- Cosine Similarity

- Recommendation Systems

# ▶️ Run Locally

## 1. Clone Repository

```bash
git clone https://github.com/your-username/pickflix-ai-movie-recommender.git
```

## 2. Open Project Folder

```bash
cd pickflix-ai-movie-recommender
```

## 3. Install Requirements

```bash
pip install -r requirements.txt
```

## 4. Run App

```bash
streamlit run app.py
```
